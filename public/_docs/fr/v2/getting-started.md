# Commencer - Version 2

Ce guide couvre l'essentiel pour v2 de notre plateforme de documentation.

## Installation

Installez v2 avec les dernières fonctionnalités :

\`\`\`bash
npm install next@latest
npm update
\`\`\`

## Fonctionnalités clés de v2

### 1. Recherche Plus Rapide

L'algorithme de recherche dans v2 est 2x plus rapide :

\`\`\`javascript
// Nouvelle configuration de recherche
const searchConfig = {
  indexing: 'optimized',
  caching: true,
  limit: 20
};
\`\`\`

### 2. Meilleur Support Hors Ligne

La documentation peut maintenant être utilisée hors ligne avec les service workers.

### 3. Expérience Mobile Améliorée

Améliorations pour les appareils mobiles :

- Navigation conviviale
- Mise en page optimisée pour mobile
- Temps de chargement plus rapides
