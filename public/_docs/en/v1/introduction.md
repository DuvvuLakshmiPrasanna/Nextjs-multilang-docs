# Introduction

Welcome to the Documentation Portal! This is a comprehensive guide to get you started with our platform.

## What is This?

This is a modern, multi-language documentation site built with Next.js. It showcases best practices in web development including:

- **Static Generation**: Pages are pre-rendered at build time for lightning-fast performance
- **Incremental Regeneration**: Content updates automatically without full rebuilds
- **Internationalization**: Full support for multiple languages
- **Dark Mode**: Seamless theme switching based on your preferences
- **Full-Text Search**: Find what you need instantly

## Getting Started

To get started, explore the documentation using the navigation menu on the left. You can:

1. Browse different versions of the docs using the version selector
2. Switch languages using the language switcher in the header
3. Toggle between dark and light modes
4. Use the search feature to find specific topics

## Navigation

- **Sidebar**: Browse documentation by page
- **Version Selector**: Switch between different versions (v1, v2, v3)
- **Language Switcher**: Change the language
- **Search**: Search across all documentation
- **Theme Toggle**: Switch between light and dark modes
